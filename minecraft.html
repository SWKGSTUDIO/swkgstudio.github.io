How to install Mod

How to install mods:

Step 1: You need to have the official version of Minecraft installed. You can download it from the official mojang website or through the following link (LINK).

Step 2: If you downloaded the mod through one of our official applications, you can run the auto-installer by clicking on the icon (ICON).

In the popup window, select "Open with Minecraft PE".

You can download our official Mods here.

If for some reason you were unable to install the mod using the automatic installation, you can do it manually. Follow the instructions below:

Step 1. Go to the file manager on your phone and locate the mod file you just downloaded (the file should have the .mcworld or .zip extension). By default, our mods are downloaded to the ../DOWNLOADS/SWKGMODS folder.

Step 2. Open the file using the Minecraft PE application. Afterward, the game will open and the mod will be automatically imported.

If none of the steps above helped or if you have any other questions, please write to us -
