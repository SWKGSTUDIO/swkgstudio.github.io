How to install Mod

How to install mods:

STEP:1
Click "Save Mod
STEP:2
Select the ../Downloads/SWKGMODS folder and click save
STEP:3
Start auto-unpacking
STEP:4
Select the folder where you downloaded the mod and click "Use this folder"
STEP:5
Don't forget to check out other games & mods ;)
STEP:6
Open files .mcworld using the Minecraft PE App. The mod import will start automatically

Download Minecraft PE

To successfully install any mod, skin, or addon, you need to download the latest official version of MCPE.
